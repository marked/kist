# kist
##Keep-It-Simple-Tivo

Custom HTTP proxy server for TivoToGo

Requires:
* perl
* tivodecode
* curl
* xmllint
* saxonb-xslt

Tested Server on:
*  Ubuntu
  
Tested Clients:
* VLC on Linux  
* VLC on Mac
* VLC Firefox Plugin

Getting started:
1. Edit 3 lines in Conf.pm
2. cd bin ; ./kistd
3. Contact server from web browser, on port 8080
